# Simple Rule-Based Chatbot

## Project Description
This is a simple chatbot built using Python. It uses a rule-based approach with pattern matching to respond to user input. The chatbot can handle basic conversations including greetings, help requests, small talk, and domain-specific questions.

## Features
- Greeting responses (e.g., hello, hi, hey)
- Help support
- Small talk (e.g., how are you)
- Knowledge base for domain questions (Python, AI, Chatbots)
- Conversation history is saved in a text file (`chat_log.txt`)

## Technologies Used
- Python
- File handling for conversation logging
- Console/terminal interaction

## How to Run
1. Install Python (if not already installed)
2. Download the repository
3. Open terminal/command prompt
4. Navigate to the project folder
5. Run the chatbot:

```bash
python chatbot.py
